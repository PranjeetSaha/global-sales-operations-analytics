# World_Bank_GDP_Analysis

## Business Objective
Build an enterprise-grade analytics platform to analyze global economic,
sales, and operational indicators for executive decision-making.

## Dataset
- Source: World Bank Open Data
- Indicator: GDP (Current USD)
- Coverage: Global, multi-year

## Skills Demonstrated
- Python & Pandas for data analysis
- SQL for business queries
- Power BI for executive dashboards
- Cloud & data engineering concepts
- Applied AI for business insights
